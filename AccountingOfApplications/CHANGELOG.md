# Changelog

Все значимые изменения в проекте «AccountingOfApplications» будут документироваться в этом файле.

## [1.0.0] - 2026-06-13

### Добавлено
- Форма авторизации с проверкой логина и пароля в Firebase
- Главное меню с навигацией по разделам
- Форма работы с заявками (просмотр, добавление, редактирование, удаление)
- Форма добавления и редактирования заявки с валидацией полей
- Форма поиска заявок по адресу или комментарию оператора
- Форма аналитики с графиками распределения заявок по типам работ и приоритетам
- Форма управления пользователями (только для администратора)
- Сортировка заявок по ID, статусу и приоритету
- Фильтрация заявок по типу работ, статусу и приоритету
- Экспорт списка заявок в форматы Excel (.xlsx) и Word (.docx)
- Автоматическая генерация уникального идентификатора заявки (МТ-001, МТ-002...)
- Валидация адреса и телефона (+7XXXXXXXXXX)
- Поддержка светлой и тёмной темы оформления
- Логирование всех действий пользователя в локальный .log файл
- Ролевая модель доступа (admin / operator / brigade)
